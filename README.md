# Ale, My Eyes!

This repository has been split into independent projects:

- [Desktop application and CLI](https://github.com/Risaly-Noroki-Dev-Club/ale-my-eyes-desktop)
- [Mobile application](https://github.com/Risaly-Noroki-Dev-Club/ale-my-eyes-mobile)
